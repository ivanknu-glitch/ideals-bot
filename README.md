# Ideals Nail Studio — Telegram Bot

## Як запустити на Railway

### Крок 1 — GitHub
1. Зайди на github.com і створи безкоштовний акаунт
2. Натисни "New repository" → назви `ideals-bot` → Create
3. Завантаж файли `bot.js` і `package.json` в репозиторій

### Крок 2 — Railway
1. Зайди на railway.app
2. "Login with GitHub" → авторизуйся
3. "New Project" → "Deploy from GitHub repo"
4. Обери `ideals-bot`
5. Railway автоматично запустить бота!

### Крок 3 — Перевірка
1. Зайди в Telegram → знайди свого бота
2. Напиши /start
3. Повинно прийти привітання як майстру

## Що вміє бот

### Для клієнта:
- Вводить код IDEALS-XXXX → прив'язується до запису
- Бачить свої записи
- Подає запит на перенесення
- Скасовує запис
- Отримує нагадування за 24 год

### Для майстра:
- Отримує нові записи з кнопками Підтвердити/Відхилити
- /bookings — всі активні записи
- /today — записи на сьогодні
- /stats — статистика
- Отримує запити на перенесення і підтверджує їх

## Підключення до сайту
Після запуску на Railway скопіюй URL сервера (виглядає як https://ideals-bot.railway.app)
і вкажи його в файлі ideals-booking.html в змінній BOOKING_API_URL
